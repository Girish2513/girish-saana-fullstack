# Shell Basics Notes

## Key Skills Learned
- Navigation using pwd, cd, ls
- File operations: touch, mv, cp, rm
- Viewing files: cat, head, tail, less
- Searching: grep
- Counting: wc
- Redirection: > >>
- Pipes: |
- Command history usage
- Creating aliases

## Important Concepts
- Commands can be chained using pipes.
- Output can be redirected to files.
- Lines are counted by newline characters.
- Shell tools can process data without writing programs.

## Safety Rules
- Avoid using rm -rf unless path is verified.
- Avoid unnecessary sudo usage.
